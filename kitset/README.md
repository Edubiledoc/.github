### NB-IoT通訊擴展板

![smaller](smaller.png)
> 可以通過Type C接口連接5V電源，或通過DC接口連接12V電源。超過12V的電源應降壓后再連接。
> 

> 開關撥至上為NBIOT模式，撥至下為MQTT模式。
> 

> 通過麵包板與麵包板兩側的排針可連接傳感器與microbit。支援5V傳感器，不支援馬達等高功率設備。
>
